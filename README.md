# Hệ thống Quản lý Nhân sự

## Giới thiệu

Xây dựng website quản lý nhân sự phục vụ quản lý thông tin nhân viên, phòng ban và tài khoản người dùng.

## Chức năng chính

- Đăng nhập hệ thống.
- Quản lý nhân viên.
- Quản lý phòng ban.
- Quản lý tài khoản.
- Tìm kiếm và cập nhật thông tin.

## Công nghệ sử dụng

- Python Flask
- MySQL
- HTML
- CSS
- JavaScript

## Cơ sở dữ liệu

- MySQL
- File SQL được cung cấp trong thư mục database.
